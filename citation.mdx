## Citing This Dataset

When using this dataset or the code used to process it, please cite the following paper:

<Note>

## Publication Details

The UXO Dataset 2024 was presented at **OCEANS 2024 - Halifax**.

**Authors**: Nikolas Dahn, Miguel Bande Firvida, Proneet Sharma, Leif Christensen, Oliver Geisle, Jochen Mohrmann, Torsten Frey, Prithvi Kumar Sanghamreddy, and Frank Kirchner

**Title**: An Acoustic and Optical Dataset for the Perception of Underwater Unexploded Ordnance (UXO)

**Conference**: OCEANS 2024 - Halifax

**Year**: 2024

**DOI**: [10.1109/OCEANS55160.2024.10754316](https://doi.org/10.1109/OCEANS55160.2024.10754316)

## Attribution Guidelines

When using the UXO Dataset 2024 in your work, please:

### For Academic Publications

1. **Cite the paper** using the BibTeX entry provided above
2. **Acknowledge the dataset** by name in your methodology section
3. **Include the Zenodo link** for dataset access

### For Software and Code

1. **Include the citation** in your README or documentation
2. **Link to the dataset** repository on Zenodo
3. **Mention the license** terms if redistributing derivative works

### For Presentations and Posters

1. **Reference the authors** and publication year
2. **Include the dataset name**: "UXO Dataset 2024"
3. **Provide access information**: Zenodo link or QR code

## Example Citations

### In-Text Citation (IEEE Style)

```text
We evaluated our UXO detection algorithm using the dataset presented in [1].

[1] N. Dahn et al., "An Acoustic and Optical Dataset for the Perception of 
Underwater Unexploded Ordnance (UXO)," in OCEANS 2024 - Halifax, 2024.
```

### Acknowledgment Section

```text
This research used the UXO Dataset 2024, available at 
https://zenodo.org/records/13778485, as described in Dahn et al. (2024).
```

### Data Availability Statement

```text
The data supporting this study are openly available from Zenodo at 
https://zenodo.org/records/13778485 [1].
```
